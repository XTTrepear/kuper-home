# 🛒 Купер Ассистент — Умное сравнение товаров

Приложение автоматически сравнивает тысячи товаров в каталоге Купера и предлагает пользователю самый лучший и выгодный вариант именно для него с учётом индивидуальных потребностей.

---

## 📦 Как запустить на Windows

### Шаг 1: Скачайте Node.js

1. Откройте https://nodejs.org/
2. Скачайте **LTS** версию для Windows (файл `.msi`)
3. Запустите установщик, нажимайте «Next» → «Next» → «Finish»
4. Проверьте установку: откройте командную строку и введите:
   ```
   node --version
   ```
   Если вывелась версия (например `v20.11.0`) — всё ок!

### Шаг 2: Распакуйте проект

1. Скачайте архив `kuper-assistant.zip`
2. Нажмите правой кнопкой → **Извлечь всё**
3. Запомните путь к папке, например `C:\Users\ВашеИмя\kuper-assistant`

### Шаг 3: Установите зависимости

1. Откройте папку с проектом
2. Нажмите в адресной строке проводника `cmd` и нажмите Enter (откроется командная строка)
   - *Или:* нажмите `Win + R`, введите `cmd`, затем `cd C:\путь\к\папке\kuper-assistant`
3. Введите команду:
   ```
   npm install
   ```
4. Дождитесь окончания (появится сообщение «added XXX packages»)

### Шаг 4: Запустите сайт

В той же командной строке введите:
```
npm run dev
```

Вы увидите сообщение:
```
VITE v5.4.21  ready in 124 ms
➜  Local:   http://localhost:3005/
```

### Шаг 5: Откройте сайт в браузере

1. Откройте Chrome / Edge / Firefox
2. В адресной строке введите: `http://localhost:3005`
3. Нажмите Enter

Готово! 🎉

## 🤖 Ассистент — главная фича

Ассистент анализирует каталог Купера и для каждого товара находит аналоги, сравнивая их по формуле:

```
Скоринг = (Цена × 40%) + (Рейтинг × 35%) + (Отзывы × 25%)
```

- 🏷️ **Лучшая цена** — самый дешёвый аналог
- ⭐ **Топ рейтинг** — товар с наивысшей оценкой
- 💬 **Больше отзывов** — самый проверенный товар
- 🔥 **Выгодно** — товар со скидкой
